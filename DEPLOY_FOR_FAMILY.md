# Как Запустить Свой Семейный Мессенджер

Эта инструкция написана для обычного человека.

Если коротко:

1. Купить VPS с Ubuntu.
2. Зайти на сервер по SSH.
3. Вставить одну команду.
4. Открыть ссылку в браузере.
5. Придумать мастер-пароль и добавить членов семьи.

## Что понадобится

- сервер `VPS` с `Ubuntu 24.04`
- `root`-доступ или пользователь с `sudo`
- 5-10 минут спокойного времени

## Шаг 1. Подключиться к серверу

Обычно хостинг присылает:

- IP сервера
- логин
- пароль

Подключение выглядит примерно так:

```bash
ssh root@123.123.123.123
```

Если система спросит `yes/no`, напиши:

```bash
yes
```

Потом вставь пароль.

## Шаг 2. Запустить установку

Скопируй и вставь на сервере одну команду:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/install.sh | bash
```

Что произойдёт дальше:

- сервер сам поставит нужные программы
- сам скачает готовый backend
- сам поднимет базу данных
- сам запустит мессенджер

Обычно это занимает несколько минут.

## Шаг 3. Открыть ссылку

В конце установки сервер покажет что-то вроде:

```text
Open in browser: http://123.123.123.123:8080
```

Скопируй эту ссылку и открой её в браузере.

## Шаг 4. Первый запуск

После открытия сайта:

1. Придумай `мастер-пароль`.
2. Добавь родителей и детей.
3. Заверши настройку.

Готово. После этого семья сможет входить в мессенджер.

## Как обновить мессенджер

Если позже выйдет новая версия, зайди на сервер по SSH и выполни:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/update.sh | bash
```

## Как удалить мессенджер

Если захочешь удалить всё с сервера:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/uninstall.sh | bash
```

## Если не открывается сайт

Проверь по очереди:

1. Дождалась ли установка конца.
2. Точно ли ты открываешь адрес, который показал сервер.
3. Не блокирует ли хостинг порт `8080`.
4. Нет ли опечатки в IP-адресе.

Проверка прямо на сервере:

```bash
curl http://127.0.0.1:8080/api/health
```

Если всё хорошо, ты увидишь ответ со словом `ok`.

## Самая короткая версия

Если уже есть сервер и SSH-доступ, нужна только эта команда:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/install.sh | bash
```
