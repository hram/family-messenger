# Пошаговый Деплой Для Семьи

Эта инструкция написана для человека, который не хочет разбираться в серверах.

Цель простая:

1. Купить сервер.
2. Вставить одну команду.
3. Открыть ссылку.
4. Настроить семью.
5. При желании скачать Android-приложение.

## Что нужно заранее

- сервер `VPS` с `Ubuntu 24.04`
- `IP` сервера
- логин и пароль для входа по `SSH`
- компьютер с интернетом

Если хостинг прислал:

- `IP`
- `root`
- пароль

Этого достаточно.

## Шаг 1. Зайти на сервер

На компьютере открой терминал.

Если у тебя:

- `Windows`:
  открой `PowerShell` или `Windows Terminal`
- `Mac`:
  открой приложение `Terminal`
- `Linux`:
  открой обычный терминал

Введи команду:

```bash
ssh root@ТВОЙ_IP
```

Пример:

```bash
ssh root@123.123.123.123
```

Если система спросит:

```text
Are you sure you want to continue connecting (yes/no)?
```

Напиши:

```text
yes
```

Потом вставь пароль сервера и нажми `Enter`.

Важно:
- когда ты вводишь пароль, символы обычно не показываются
- это нормально

## Шаг 2. Запустить установку

После входа на сервер скопируй и вставь ровно эту команду:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/install.sh | bash
```

Дальше ничего делать не нужно.

Сервер сам:

- поставит нужные программы
- скачает готовую версию мессенджера
- поднимет базу данных
- запустит сайт

Обычно это занимает несколько минут.

## Шаг 3. Дождаться конца установки

Когда всё закончится, на экране появится строка такого вида:

```text
Open in browser: http://123.123.123.123:8080
```

Это и есть твоя ссылка на мессенджер.

Скопируй её и открой в браузере.

## Шаг 4. Первый запуск

Когда откроется сайт:

1. Придумай `мастер-пароль`.
2. Введи его два раза.
3. Нажми `Next`.
4. Введи название семьи.
5. Добавь родителей и детей.
6. У одного из родителей поставь галочку `Administrator`.
7. Заверши настройку.

После этого система создаст коды входа для членов семьи.

## Шаг 5. Войти в мессенджер

После настройки:

- открой сайт снова
- введи код входа нужного члена семьи
- нажми кнопку входа

## Шаг 6. Скачать Android-приложение

Если Android-приложение уже выложено на сервер, оно будет доступно по ссылке:

```text
http://ТВОЙ_IP:8080/downloads/family-messenger-android-no-fcm-debug.apk
```

Пример:

```text
http://123.123.123.123:8080/downloads/family-messenger-android-no-fcm-debug.apk
```

Что сделать:

1. Открой эту ссылку на телефоне.
2. Скачай файл.
3. Разреши установку из браузера, если Android попросит.
4. Установи приложение.

Если приложение уже открыто со старым адресом сервера:

- открой настройки приложения
- поставь правильный адрес сервера:

```text
http://ТВОЙ_IP:8080
```

## Как обновить мессенджер

Позже, когда появится новая версия, снова зайди на сервер по `SSH` и выполни:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/update.sh | bash
```

## Как удалить мессенджер

Если захочешь полностью всё удалить с сервера:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/uninstall.sh | bash
```

## Если сайт не открывается

Проверь по порядку:

1. Установка точно закончилась до конца.
2. Ты открываешь именно ссылку, которую напечатал сервер.
3. В адресе нет опечатки.
4. Хостинг не блокирует порт `8080`.
5. Открой сайт в приватной вкладке браузера.

Если хочешь проверить сам сервер, выполни по `SSH`:

```bash
curl http://127.0.0.1:8081/api/health
```

Если всё в порядке, ты увидишь ответ со словом `ok`.

## Самая короткая версия

Если у тебя уже есть сервер и доступ по `SSH`, то главная команда только одна:

```bash
curl -fsSL https://raw.githubusercontent.com/hram/family-messenger/main/infra/install.sh | bash
```
